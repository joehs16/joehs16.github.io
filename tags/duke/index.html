Demo of Azure Normalize Pipeline

By Joe Hsieh |  Mar 9, 2021  | duke, themes, development, etl
Demo of AzureNormalizePipeline Last edited: Mar 05, 2021 Description This function was build in support of the BIGIDEAS Lab’s Covidentify Project (Duke University). The intention of the function is to extract, transform, and load the unstructured json data from the database that houses the Fitbit and Garmin API pulls into Microsoft Azure’s ML Studio. Background Because of the nature of Duke’s Internal Review Board (IRB), the lab does not have direct control over the API pulls and thus using noSQL was not an option at this time.
